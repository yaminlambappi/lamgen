JSON formatting (also called JSON beautifying or pretty-printing) adds proper indentation and line breaks to compact JSON, making it human-readable. It does not change the data — only the whitespace.
Formatting adds indentation for readability. Minifying removes all whitespace to reduce file size — useful for production APIs and web performance. Both produce valid JSON with identical data.
