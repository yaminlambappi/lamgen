JPEG and WebP use lossy compression, which slightly reduces quality to achieve smaller file sizes. At 80% quality the difference is usually invisible to the human eye. PNG uses lossless compression — no quality loss but larger files.
WebP typically achieves 25–35% smaller file sizes than JPEG at equivalent quality. It is supported by all modern browsers and is the recommended format for web images.
